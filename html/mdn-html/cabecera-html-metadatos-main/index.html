Cabecera de HTML: Metadatos

La etiqueta head es la parte no visible de una página que contiene otras etiquetas/elementos para las hojas de estilo (CSS), icono de la web que aparece en la pestaña del navegador y metadatos que son datos que describen el contenido de la web que ayudan a posicionar el sitio en los buscadores.

Elementos principales de la cabecera

En sitios grandes, la cabecera puede tener muchisimos elementos y metadatos para describir a nuestro sitio, poner el autor, resumirlo, etc. Para empezar repasaremos los esenciales.

Etiqueta meta

El elemento meta solo posee una etiqueta de apertura donde se ubicarán los atributos con los que describiremos al sitio.

En la etiqueta meta encontraremos atributos en comun como name en donde va el nombre del metadato que vamos a usar y content que es contenido/texto del metadato.

  • meta charset: Determina los caracteres de un idioma conocido que aceptará nuestra web. Se usa el valor utf-8 para asegurarnos que nuestro sitio muestre todos los caracteres del mundo, sean arabes, japoneses; cuando los usuarios tengan que interactuar con nuestra web.
  • meta author: En el atributo content debemos colocar el nombre de nuestra web.
  • meta description: En content debemos escribir el objetivo/proposito de nuestro sitio/página, cuál es su contenido, sus ventajas, etc. Aparece en el buscador web, debajo del titulo de la página.
  • meta keywords: Palabras claves para nuestra página que el buscador web las tomará en cuenta a la hora de dar un sus resultados. Aunque este elemento es ignorado hoy en día por los buscadores dado a que se usaba para spamear palabras así conseguir mejor ubicación en los resultados.
  • meta viewport: Esto tiene que ver con la visualización de nuestra página en los navegadores moviles. Este tema se tratará al final del curso de CSS.

Etiqueta title

Se usa para poner el titulo de la página que va a aparecer en la pestaña del navegador. Ademas cuando guardamos una página en marcadores, el titulo de sugerencia es el que aparece en la pestaña.

Etiqueta Link

Link se usa para anclar un recurso externo o interno a nuestro documento HTML. Un recurso interno puede ser una hoja de estilo que este en alguna carpeta en nuestro proyecto. Un recurso externo puede ser una fuente de google usada en alguna parte de nuestro proyecto.

Dentro de link generalemente se encuentran los atributos rel que describe a que se refiere el atributo href que determina la dirección del recurso/archivo atraido por esta etiqueta.

Agregar iconos al sitio

Son iconos de 16x16 en formato .png, .gif y .ico que se mostrarán en la pestaña abierta o cuando guardamos una página en marcadores. Este icono llamado favicon se puede usar en diferentes contextos que requieran uno de mayor resolución como los que se usan en dispositivos como ipads, etc.

Usamos el elemento link en el head (parte no visible del html) donde es más usado. Vemos los atributos rel y href explicados anteriormente. Link representa otro tipo de enlace junto con el elemento a pero es más usado en la parte visible/contenido de la web (body), la diferencia es proposito o la relación con el recurso que enlaza como se explica en tipos de enlace. En conclusión el elemento link va hacia recursos y el elemento a va hacia otras paginas.

Los enlaces tienen su particularidades porque debemos describir el atributo rel en funcion de su recurso (href), como se explica en tipos de enlace (aunque en el elemento a no se suele usar el atributo el).

Junto con los iconos son otro topico de HTML que se ampliará más adelante.

CSS y JavaScript en el head

Se aplica CSS para darle una buena disposición y aspecto a los elementos del HTML, además se agrega JS para darle una buena funcionalidad.

Se puede agregar estos dos desde el head con los elementos style y script.

Aunque no se recomienda este metodo para un sitio profesional, por el contrario se usa el elemento link para CSS y el elemento script para JS. Los dos elementos dependen de un archivo o recurso externo al HTML como .CSS que va en link y .js que va en script. La diferencia esta en que link va en la cabecera pero script va al final del body HTML para que carge la estructura y el aspecto del sitio antes que sus funciones así carga más rapido.

  • El elemento link tiene los atributos rel="stylesheet" que es la referencia al recurso que está en href y href="style.css" es la dirección de la hoja de estilo.
  • El elemento script puede tener el atributo type="text/js" y el atributo src="script/main.js" aquí va la direccion donde esté la hoja con el código de javascript.

Vemos que los atributos siempre estan en la etiqueta de apertura.

Establecer el idioma principal del sitio

Se hace añadiendo el atributo lang en el elemento HTML. Los motores de busqueda lo usarán para dar los resultados del sitio en el idioma respectivo. Tambien es usado por los lectores de pantalla (para usuarios no videntes) para que pronuncien las palabras adecuadamente según el idioma.

Con el atributo lang podemos establecer una parte textual de la página que este en otro y se interprete como este. Por ejemplo, si una palabra está en inglés y usamos lang en está; un lector de pantalla lo va a leer como se debe en este idioma.
