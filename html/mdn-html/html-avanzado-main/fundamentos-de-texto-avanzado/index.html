Marcado de texto avanzado

Existen una variedad de elementos para el marcado de texto que podemos usar para las citas, código de programación, subíndices, superaíndices e información de contacto para los visitantes del sitio.

Citas

Hay citas que se pueden marcar dentro de los parrafos o un elemento en bloque, llamadas citas en línea y hay citas que son un elemento en bloque directamente, llamadas (citas en bloque).

Blockquote: cita en bloque

El elemento blockquote tiene dentro de su etiqueta de apertura el atributo cite en el que va el texto. La cita empezará en una linea nueva como todos los elementos de bloque.

Elemento q: cita en línea

El elemento q tambien posee el atributo cite. Va dentro de un parrafo u otro elemento en bloque, la cita de mostrará dentro de ellos sin hacer un salto de línea.

Elemento cite: titulo del autor

El elemento cite se usa para dar el autor de una frase, el nombre del libro, material o recurso que se uso para la cita. Su aspecto aspecto por defecto se suele mejorar con CSS, es un elemento en línea.

Abreviaturas

El elemento abbr se usa para escribir una abreviatura, dentro de su etiqueta de apertura está el atributo title que se usa para poner el significado completo de esta. El texto de title se ve cuando pasamos el cursor por encima de la abreviatura.

Marcar información de contacto

Con el elemento adress incluimos toda la información de contacto que puede necesitar un cliente/visitante, como el nombre de la empresa, ubicación telefono, mail de contacto o enlaces.

Podemos escribir todo dentro de un parrafo o usar una lista no ordenada para lograr otra visualización.

Superíndice y subíndice

Los elementos y se usan para marcar superíndices y subíndices en fechas, fórmulas quimicas, ecuaciones matematicas para que sean visualmente correctas.

Marcar código de programación

Se usan una serie de elementos de HTML para que el código pueda ser presentados sin conflictos con el lenguaje de programación en el editor de texto.

  • code: Se usa para escribir el código
  • pre: Se usa para poder usar espacios en el elemento code, por lo tanto encerrará a este elemento. Son espacios que tendría cualquier lenguaje, de lo contrario el navegador ignoraría los espacios.
  • var: Para resaltar los nombres de las variables.
  • kdb: Para marcar combinaciones de teclado que tengan algun efecto como copiar, pegar, seleccionar todo el texto.
  • samp: Para marcar la respuesta que da una terminal, consola de un lenguaje de programación; pueden dar errores, caracteristicas, descripciones, instrucciones, etc.

Marcar horas y fechas

Para que el navegador reconozca una fecha (dia/mes/año en cualquier combinación) se usa el elemento time. Tambien funciona para las horas, minutos y segundo.

Dentro de la etiqueta de apertura se encuentra el atributo datatime en el que ponemos un formato de fecha u horario para que la maquina lo reconozca. Dentro de las dos etiquetas va la fecha u horario para que el usuario la puede ver.
