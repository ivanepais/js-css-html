Marcado de tablas

Con HTML podemos crear tablas de datos, nos brinda elementos para hacer celdas, filas, encabezados; tambien metodos para que se puede abarcar multiples celdas o filas.

¿Que es una tabla?

Es un conjunto de datos distribuidos en filas y columnas, conocidos como datos tabulados. Permite ver, buscar y comparar datos con rapidez, el conjunto de datos permiet hacer algun tipo de conexión entre lo que se representa.

Dar estilo a una tabla

Por defecto tienen un estilo minimo que les da el navegador pero para hacerlas más efectivas se puede aplicar estilo a las tablas con CSS.

Cuando no usar una tabla

No se debe usar para crear diseños web y volcar el contenido en ella. La confusión viene porque una web y una tabla tienen cabecera, filas y columnas para introducir contenido y pie de página.

Las tablas tiene menos capacidad para dar accesibilidad para los usuarios, no tienen la estructura necesaria para hacer una web. Ademas, las tablas no son responsivas por defecto, necesitan medidas adicionales para que se adapten al tamaño del contenido y pantalla.

Elementos para crear una tabla

El contenido de la tabla esta delimintado por el elemento table. Para crear una celda dentro de la tabla se usa el elemento td, es la unidad más pequeña de una tabla. Si queremos crear una fila tenemos que tener varios elementos td.

Para deliminar una fila ("detener su crecimiento"), agrupamos todos los elementos td en el elemento tr.

Para hacer encabezados semantica (SEO) y visualmente a algunos elmentos usamos th, se usan para distinguir o destacar la información y hacer una conexión con los datos.

Agrupar datos en filas o columnas

Se usan para categorizar los datos.

En los encabezados de las filas usamos el atributo rowspan. Para las columnas colspan. Necesitan de un valor que será un numero natural, este determina cuantas celdas abarcará el encabezado fila o columna.

Elementos para aplicar estilo a las columnas

Necesitamos colgroup para agrupar los elementos de la columnas y usamos col con el atributo style para darle estilo a todas estas columnas y no repetir una por una.

Funcionamiento de colgroup y col: Podemos ubicarlos al principio de la tabla, despues de crear todas las columnas; porque colgroup contiene la cantidad de columnas que estan representadas por col, que es solo una etiqueta de apertura. En ella va el atributo style.

Por ejemplo, si tenemos dos columnas, hay dos col y si solo queremos aplicar un estilo a todos los elementos de la segunda columnas, se lo aplicamos al segundo col.
