Alineación en flex

La especificación Box Alignment explica como funcionan las alineaciones en todos los metodos de diseño, lo que incluye a flexbox. Los detalles se especifican en CSS Box Alignment.

Un ejemplo

Los elementos se alinean en el eje principal con justify-content y para alinear elementos en el eje transversal (corta al eje primario) se usa align-content.

.box { display: flex; align-items: flex-start; justify-content: space-between; } .box :first-child { align-self: center; }

Flex está atado a los modos de escritura, estos modifican y cambian el comportamiento de los ejes.

Los ejes y flex-direction

Podemos cambiar el eje pricipal configurando flex-direction a column, por defecto viene con el modo de escritura en ingles que muestra el contenido de izquierda a derecha; que sería flex-direction: row.

  • El eje principal sigue a flex-direction que sigue a justify-content.
  • El eje transversal corta al principal y alineará los elementos con align-content, align-self/content/items.

No exite la propiedad justify-self en Flexbox

Flexbox trata el contenido en conjunto, calcula el espacio necesario para los elementos y el espacio sobrante para distribuirlos. La propiedad justify-content controla como se usa el espacio sobrante.

En el eje transversal tenemos espacio para align-self para mover los elementos desde el principio hasta el final.

La propiedad justify-self no tiene sentido en flex ya que siempre tratamos de mover

Alineación y auto margenes

Si queremos dividir elementos para crear una navegación dividida podemos usar auto margin que absorverá todo el espacio disponible en su dimension, así es como se hacia para el centrado de bloque; poniendolo en ambos lados (absorviendo el espacio al costado), empujando la caja al centro. Fue remplazado por flex.

Si establecemos auto margin a un elemento empuja hacia un lado los elementos que siguen a este. Cuando no hay espacio disponible para el margen automatico, flex actúa de la misma manera; reduciendo espacio para que entre todos los elementos.

.box { display: flex; } .push { margin-left: auto; }

A partir del cuarto elemento son empujados hacia la izquierda

Propiedades para crear espacios

Son row-gap, column-gap y gap como abreviada.

Crear espacios de tamaño fijo entre elementos

En el eje principal, la propiedad column-gap crea espacios de tamaño fijo entre elementos adyacentes (a los lados).

En el eje transversal, la propiedad row-gap crea espacio en las lineas adyacentes, debe haber flex-wrap.
