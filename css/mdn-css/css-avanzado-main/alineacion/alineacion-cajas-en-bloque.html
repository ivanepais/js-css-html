Alineación de cajas en bloque, posición absoluta y diseño de tablas

La especificación de Box Alignment determina como funciona la alineación para cada metodo de diseño. Empezamos por la alineación en el contexto de bloque que incluyen los elementos posicionados, flotantes y de tabla. Debe leerse junto con la página CSS Box Alignment que detalla las propiedades para todos los metodos de diseño.

align-content y justify-content

La propiedad justify-content no se aplica a contenedores de bloque o celdas de tablas.

La propiedad align-content al eje en bloque (o dirección en bloque) para alinear el contenido de la caja dentro que está dentro del contenedor. Si el contenido necesita de space-between, space-around o space-evely; requieren de una alineación alternativa porque el contenido será tratado como un sujeto de alineación unico.

justify-self

Alinea un elemento dentro del bloque contenedor en el eje en linea (o dirección de contenido).

align-self

No se aplica a las cajas en bloques, incluidos los flotantes; porque hay más elementos en el eje de bloque. Tampoco se apliqua a las celdas de las tablas.

Elementos con posicion absoluta

El bloque posicionado se mueve con las propiedades top, left, right y bottom.

Alineación en los metodos de diseño hoy

No tenemos alineación compatible para los bloques, por eso se usa flexbox u otro metodo para alinear un grupo de elementos.

La alineación de bloques en horizontal antes de que exista flexbox, se hacia con margenes automaticos. Un margen en auto absorve el espacio, si lo usamos a la izquierda y derecha; empuja el bloque hacia el centro.

.container { width: 20em; margin-left: auto; margin-right: auto; }

En el diseño de tabla se usa vertical-align para alinear el contenido de una celde de la tabla.

Para muchos casos de uso, podemos convertir el contenedor de bloques en un contenedor flexible para alinear horizontalmente. En el ejemplo, tenemos un contendor flex con un solo elemento: dentro del div (.box) hay un sub elemento div.

.box { display: flex; align-items: center; justify-content: center; }
