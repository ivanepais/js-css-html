Alineación de caja en Grid

Como antes, la especificación CSS Box Alignment detalla la alineación que usamos para el diseño de cuadrículas.

Un ejemplo

En grid, el eje principal es la dirección en línea y el eje transversal es la dirección en bloque por donde corren los elementos. La alineación de los elementos del contenedor se controla con align-items para el eje transversal (bloque) y justify-content para el eje principal (línea).

.box { display: grid; width: 540px; grid-template-columns: 120px 120px 120px; align-items: start; justify-content: space-between; } .box :first-child { align-self: center; }

Ejes en Grid

En las dos dimensiones de grid tenemos disponibles todas las propiedades de alineaciones disponibles para los elementos.

El eje en línea es la dirección dónde corre el texto que cambia con el modo de escritura, en ingles son horizontales. Se usan las propiedaes justify-content/items/self.

El eje en bloque es la dirección hacia dónde aparecen los elementos, por ejemplo, en ingles; los parrafos aparecen uno debajo de otro, es vertical. Se usan las propiedades align-content/items/self.

Alineación individual

  • justify-self
  • align-self
  • place-self
  • justify-items
  • align-items
  • place-items

Se ocupan de alinear los elementos en las areas de las cuadrícula.

Las propiedades con la palabra items se aplican al contenedor y con la palabra self se aplican a los elementos individuales. Podemos establecer una alineación para todos los elementos de la cuadrícula y después tomar un elemento individual y cambiar su alineación.

El valor inicial de las propiedades self es stretch, el elemento se extenderá por toda el área de la cuadrícula, a menos que tengamos un elemento media (img, vid, etc) con una relación de tamaño intriseca que se mostrará tal cual para que no se distorsione.

Content alignment

  • justify-content
  • align-content
  • place-content

Son para alinear las pistas de la cuadrícula cuando hay espacio para distribuir, sucede cuando todas las pistas suman menos que el total del contenedor.

Gap y gap heredado

  • row-gap
  • column-gap
  • gap

Estas se usan tambien en otros metodos de diseño que tengan sentido el espacio sobrante.

Si un navegador no tiene compatibilidad para estas, se usan las propiedades heredadas: grid-row-gap, grid-column-gap y grid-gap.
