Usando la ubicación basada en las líneas de Grid

Las líneas estan numeradas tanto para las filas como para las columnas, empiezan en 1 y esto depende del modo de escritura

Ejemplo basico

Si ponemos 3 filas y tres columnas nos dan 4 líneas para cada dimensión. Se ubican automaticamente si no le damos nosotros, una ubicación distinta a todos los elementos.

En el inspector podemos activar una opción para que nos muestre las líneas de grid.

Posicionar elementos de acuardo a los numeros de líneas

El primer valor será la línea de inicio y el segundo será el de la línea final para la columna y la fila. El elemento puede tener una forma horizontal o vertical.

Los demás elementos se mueven automaticamente. Tambien podemos dejar celdas vacias, evitando las líneas que definen ese espacio.

Abreviaturas para colocar elementos: grid-column y grid-row

Los valores de inicio y fin se colocarán mediante una barra (/).

.box1 { grid-column: 1 / 2; grid-row: 1 / 4; }

Expansion de elemetos por defecto

Si un elemento abarca toda la fila o columna, solo tenemos que poner el valor de inicio.

Podemos replicar el ejemplo anterior con menos valores, solo con los de inicio.

.box1 { grid-column-start: 1; grid-row-start: 1; grid-row-end: 4; }

Con esta misma tecnica podemos dejar un espacio vacio

Propiedad grid-area

Con esta solo propiedad podemos poner todos los valores de columna y fila en un solo renglon. Pero tiene un orden:

  1. grid-row-start
  2. grid-column-start
  3. grid-row-end
  4. grid-column-end

Inspiradas en el modo de escritura y en las relativas al flujo de texto y cajas, o sea; hacia donde va el contenido y las cajas, llamadas propiedades lógicas:

  • block-start
  • block-end
  • inline-start
  • inline-end

Primero definimos ambas líneas de inicio block-start e inline-start, luego ambas líneas finales block-end e inline-end.

.box1 { grid-area: 1 / 1 / 4 / 2; }

Colocar elementos a la inversa: líneas negativas

Tambien depende del modo de escritura y la direccion del bloque (orden aparición/visualización) y en línea (texto/contenido), la reversa en inglés empieza de derecha a izquierda. La primera es -1, la segunda es -2, etc.

Las líneas negativas no aparecen en las filas o columnas implicitas (creadas automaticamente).

.box1 { grid-column-start: -1; grid-column-end: -2; grid-row-start: -1; grid-row-end: -4; }

Los elementos aparecen del otro lado, respecto al anterior ejemplo.

Estirar los elementos a lo largo de la cuadrícula

Podemos combinar los numero de linea positivos y negativos

.item { grid-column: 1 / -1; }

Espacio entre elementos

Los espacios entre las filas y las columnas actúan igual que en el diseño de multicol, restan espacio a las filas y columnas para no agrandar al contenedor.

Se usan las propiedades colum-gap, row-gap o gap que usa un valor las dos o podemos usar dos valores, uno para cada una.

Usando la palabra clave: span

Con esta palabra + un numero, podemos estirar las líneas de inicio y fin al elemento.

.box1 { grid-column: 1; grid-row: 1 / span 3; }
