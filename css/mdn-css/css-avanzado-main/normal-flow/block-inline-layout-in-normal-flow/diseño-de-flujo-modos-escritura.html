Anidación de modos de escritura

Cuando a un elemento/caja anidada tiene un modo de escritura distinto al de su padre:

  • Un elemento en línea se mostrará como si tuviera display: inline-block.
  • un elemento en bloque creará un nueva BFC, a su vez, éste determina que la visualizacion interna del elemento estará fuera del flujo.

Elementos de reemplazo (img, video, etc)

No cambian su orientación/visualización por el modo de escritura. Los elementos que controlan un formulario que tienen texto, deben coincidir con el modo de escritura de uso.
