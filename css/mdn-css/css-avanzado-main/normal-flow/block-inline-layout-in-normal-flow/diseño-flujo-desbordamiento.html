Controlando el desbordamiento (overflow)

Hay valores alternativos para controlar el contenido desbordado que no son los más conocidos.

La propiedad overflow es una abreviatura de overflow-x y overflow-y.

Propiedades logicas relativas al flujo

Vimos en direcciones de texto alternativas a horizontal-tb podemos usar nuevas propiedades (logicas) como block-size e inline-size que reemplazan a propiedades fisicas height y width.

En la especificación de nivel 3 de Oveflow se incluyeron las propiedades logicas overflow-block y overflow-inline, tambien para usarlas en writting-mode distinto de horizontal-tb.

Corresponden a las direcciones overflow-x y overflow-y pero como son lógicas dependen de lawritting-mode de los elementos. Aunque todavía no están implementadas en los navegadores.

Debemos seguir usando las propiedades fisicas comunes de overflow y cambiar el writting-mode.

Indicando desbordamiento

En la especificación de overflow de nivel 3, hay propiedades para mejorar la apariencia del contenido.

Desbordamiento horizontal en una caja

Si teneemos una caja que tiene ancho limitado y no podemos hacer que el contenido de texto sea responsivo/no siga en otra línea.

.box p {white-space: nowrap; text-overflow: ellipsis; overflow: hidden;}

La propiedad text-overflow se ocupa del desbordamiento de texto en la dirección del texto/contenido, segun writting-mode.

Tiene como valores a clip que es el predeterminado, corta/esconde el contenido que no entra en la caja. El otro valor es ellipsis que pone puntos suspensivos antes de esconder el contenido.

Desbordamiento vertical en una caja

La propiedad block-overflow es una propuesta (a la comunidad que determina los estandares de CSS) que con el valor ellipsis Permitiría agregar puntos suspensivos cuando el contenido se desborda verticalmente. Por ejemplo, en una lista de articulos dado que son cuadros pequeños con altura fija con poca capacidad para texto.

Estas propiedades funcionan para medios continuos (web, video, etc) y medios páginados (documentos para imprimir, pdf).
