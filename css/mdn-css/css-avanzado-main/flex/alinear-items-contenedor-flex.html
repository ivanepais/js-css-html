Alinear elementos en un contenedor flexible

La alineación de los elementos depende del valor de direccion que tome el contenedor, si es columna o fila; cambiarán los ejes.

Si no determinamos una direccion con la propiedad flex, podemos usar las dos propiedades de alineacion que dispone el contenedor para los dos ejes: align-items y justify-content

Propiedades que controlan la alineación

  • justify-content: Controla la alineación de todos los elementos en el eje principal (main).
  • align-items: Controla la alineación de todos los elementos en el eje secundario (cross).
  • align-self: Controla la alineación de un elemento en el eje secundario.
  • align-content: Controla el espacio entre las líneas del eje secundario.
  • gap, column-gap, and row-gap: Crean espacios entre elementos flexibles.

Valores

Align-items: En eje secundario

  • flex-start
  • flex-end
  • center
  • stretch
  • baseline

justify-content: En eje principal

  • flex-start
  • flex-end
  • center
  • space-between
  • space-around
  • space-evenly

align-content: En eje secundario

  • flex-start
  • flex-end
  • center
  • space-between
  • space-around
  • stretch
  • space-evenly

Usar margenes automaticos para alinear items en el eje principal

Hacemos una alineación individual para separar un elemento o un grupo de elementos

Un ejemplo comun se da en una header con botones para la navegación en el sitio, el grupo principal puede estar a la izquierda y el secundario a la derecha. Puede pensar que están alineados con justify-self, pero no; dado que tenemos tres elementos de un lado y dos del otro. Cambiar la alineación del elemento "d", cambia tambien la del elemento "e".

En vez de esto, podemos apuntar al elemento "4" y separarlo de los primeros tres elementos, dandole un valor de margen izquierdo auto, ocuparán todo el espacio que pueda en su eje. Así funcionan los margenes automaticos para la izquierda y derecha.

Vemos dos elementos a la izquierda o al final que están separados de los del inicio.

Tenemos dos clases, .box para flex basico y .push para aplicar margen auto a la izquierda.

Crear espacios entre elementos flexibles

Con las propiedades gap (abreviada), column-gap (para el eje principal) y row-gap (entre lineas flexibles, cuando tenemos wrap -elementos en varias lineas-).
