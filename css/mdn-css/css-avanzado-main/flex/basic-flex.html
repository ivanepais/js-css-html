Flex direction

Basicamente, cada dirección tiene dos ejes. Eje principlay y eje vertical.

  • Row: el eje principal es la direción en línea y el vertical es la direccion en bloque.
  • Column: el eje principal es la dirección en bloque y el vertical es la dirección en línea.

Tambien tienen la ubicación dónde empiezan o terminan los elementos (start/end).

Flex abreviada

La abreviatura flex: 1; o flex: 2; es como si usará flex: 1 1 0 o flex: 2 1 0; para los valores flex grow (crecimiento), shrink (reducción) o basis (tamaño). Los elementos pueden crecer o reducirse a partir de una base flexible de 0.

El valor auto indica que los elementos toman el tamaño establecido en ellos mismo o si no tienen, su contenido determinará su tamaño.

El valor 0 indica que va a ignorar el tamaño "auto" de los elementos (por contenido o por su ancho definido) así comparte todo el espacio disponible (del contenedor) proporcionalmente.

Otras:

  • flex: initial resetea a los valores iniciales que son flex: 0 1, cero para crecimiento, uno para reducción y auto para la base de los valores anteriores.
  • Flex: auto es flex: 1 1 auto, los elementos pueden crecer y reducirse.
  • Flex: none es Flex: 0 0 auto se distribuyen en el contenedor utilizando el tamaño mismo de los elementos, debido al valor auto.

Alinear elementos flexibles

A donde empiezan o terminan los elementos, está determinado por la propiedad writting-mode.

Las propiedades para alinear los elementos flexibles van en el contenedor flexible. Los ejes (main/principal; cross/secundario) son importantes, cambian respecto a si hacemos una fila/row (main es horizontal y cross es vertical) o una columna/column (main es vertical y cross es horizontal).

Para el eje secundario usamos las propiedades align-items y para el eje principal justify-content.
