Capitulo 4: Fuentes web

Son tipos de letra como los que hemos hablado en fundamentos para el texto pero se detallará el uso de las tipografias web para lograr un estilo más personalizado.

Introducción a las fuentes web

Con font-family podemos aplicar una lista de fuentes y el navegador buscará la que tenga el sistema y se mostrará. Hay un conjunto de fuentes que están en todos los sistemas: Arial (sans-serif), Courier New (monospace), Georgia (serif), Times New Roman (serif), Trebuchef MS (sans-serif), Verdana (sans-serif). Tambien hay fuentes predeterminadas: serif, sans-serif, monospace, cursive, fantasy. El problema de este metodo es que puede desencajar con tus diseños.

Web fonts

Es una función de CSS que adminte archivos de fuentes para que se descargen cuando el usuario accede al sitio, para no depender de las fuentes que dispone el sistema.

En la primera linea del CSS se declara la funcion @font-face, usamos la propiedad font-family para nombrar la fuente y la propiedad src con la funcion url() y la dirección donde se encuentra nuestra fuente.

En otra regla, tomamos el HTML y usamos font-family para dar una listas de fuentes, poniendo el nombre de @font-face y otras fuentes para personalizar el sitio.

Para usar este metodo hay que tener en cuenta que:

  1. Los navegadores adminte diferentes formatos de letras, por ejemplo, los modernos adminten WOFF/WOFF2 (Web Open Font Format 1 y 2). Las versiones antiguas de IE admiten EOT (Embedded Open Type). Para navegadores antiguos de Iphone y Android se usa SVG.
  2. Las fuentes pueden tener un costo y/o respetar las condiciones de la licencia como puede ser otorgar los creditos al creador.

Encontrar Web fonts

  • Para fuentes gratuitas: Font Squirrel, dafont, Every Fonts en las que se debe respetar las condiciones de uso.
  • Para fuentes de pago: fonts.com, myfonts.com. Tambien puedes comprar fonts de estudios de diseño como Linotype, Monotype, Exljbris.
  • Hay sitios de servicios de fuentas: almacenan fuentes y facilitan el proceso.

Aplicar una fuente a un sitio: Practica

Hay que tener en cuanta los conceptos anteriores y seguir los pasos detalladamente.

Bajamos los archivos HTML y CSS de ejemplo para practicar de forma local.

Vamos a Font Squirrel a elegir dos tipos de letra, una para encabezados (puede ser de tipo Display o Slab serif) y otra para parrafos (menos llamativa y comoda para leer como sans). Descargamos el archivo, no importa el formato (TTF o OTF) y lo guardamos en la carpeta del HTML y CSS, descomprimir el zip para ver y elegir las fuentes.

El archivo generalmente zip, puede tener varias versiones de una misma fuente (como thin, regular, normal, bold, etc) y archivo de licencia.

Generar el código

Necesitamos un codigo para aplicar la fuente web, siguiendo:

  1. Cumplir con la licencia, sobre todo si es para uso comercial.
  2. Ir a Webfont Generator de Font Squirrel o de las demas páginas.
  3. Subir las dos fuentes que elegimos, con el boton Upload Fonts.
  4. Marcar la casilla de verificación para usar las fuentes gratuitas.
  5. Descargar el kit.

Cuando el generador haya terminado, deberas descargar el archivo zip y guardarlo en la carpeta donde esta el HTML y el CSS.

Aplicando fuentes a la web

Una vez descomprimido el zip, vemos varios archivos utiles:

  • Para una misma fuente vemos hay versiones con distinta extensión por ejemplo: .ttf, woff, woff2 que se necesitan para tener compatibilidad con los navegadores.
  • Un HTML de ejemplo para cada fuente así vemos como queda en uso.
  • Una hoja de estilo CSS que contiene el código de la propiedad @font-face que se necesita.

Implementar las fuentes:

  1. Cambiar el nombre de la carpeta por fonts para que sea más facil encontrarla en el proyecto.
  2. De la hoja de estilo de las fuentes se copia el codigo @font-face en nuestra hoja de estilo de practica (o en la de algun proyecto) al principio antes de las demás reglas
  3. El codigo que copiamos hay funciones url(), ahí ponemos la direccion de nuestra carpeta de fuentes: fonts/ seguido del nombre de los archivos de fuente.
  4. Ahora podemos usar estas fuentes en la propiedad font-family con el nombre de la fuente descargada y hacer una lista.

Con las demas propiedades de CSS para las fuentes podemos ajustarlas para darle un mejor aspecto del que viene determinada.

Tambien podemos convertir los tipos de fuente en otros, por ejemplo de ttf a wolf.

  • Con el repo sfnt2woff convertimos el tipo ttf a woff.
  • Con fontforge convertimos ttf a svg.
  • Con batik convertimos ttf a svg.
  • Con woff2 convertimos ttf a woff2.

Servicios de fuentes online

Proporcionan fuentes con tan solo copiar unas lineas de codigo sin utilizar @font-face. Hay servicios de pago como: Adobe Fonts y Cloud.typography y gratuitos como Google Fonts.

Podemos usar el servicio de google en los ejemplos de practica anteriores.

  1. Podemos seleccionar varios tipos de letra con el botón +.
  2. En el icono superior derecho desplegamos un menú para copiar el código HTML que va en la etiqueta head y el codigo CSS para nuestra hoja de estilo.
  3. Podemos personalizar estas fuentes con las propiedades adecuadas.
  4. Si usamos las hojas de practica anteriores tenemos que comentar las líneas correspondientes a @font-face y cambiar el nombre de los tipos de letras.

@font-face al detalle

Algunas propiedades que podemos encontrar en su declaración:

  • font-family: Deberias especificar el nombre de la fuente que estas usando, es recomendable que mantengas los tipos de fuentes para cada sección del sitio.
  • src: Se definen las rutas de acceso a los archivos o recursos. Aparecen como funciones como url() format() en donde se pone la ruta y el tipo de archivo para compatibilidad. Vemos que podemos poner varias declaraciones separadas por comas. Para versiones viejas de IE se colocan primero las fuentes de formato EOT.
  • font-weight y font-style: Se trata del grueso y la cursiva de la letra para los elementos de texto. Cuando importamos variaciones de una fuente definimos su stylo y grueso con el mismo nombre de fuente.

Fuentes variables

Permiten incorporar variaciones a la letra en un mismo archivo sin necesidad de tener varios archivos con una misma fuente pero de distinto aspecto (como cuando descargamos fuentes de Font Squirrel) a traves de CSS y la propiedad @font-face.

Se logran fuentes extremadamente personalizables que permite explotar la creatividad. Puedes aprender más en la extensa guía sobre fuentes variables en MDN.
