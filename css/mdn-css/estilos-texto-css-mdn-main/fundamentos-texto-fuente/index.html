Fundamentos de texto y fuentes tipograficas

Los fundamentos de su diseño son el grosor (font-weight), familia (font-family) y estilo de letra (font-style), las propiedades abreviadas, la alineación del texto (text-alignment), espaciado entre línea (line-height) y espaciado entre letras (letter-spacing) entre otras propiedades.

Aplicar estilo al texto

El texto del elemento dentro de una caja empieza en la parte superior izquierda en caso de usar la dirección de izquierda a derecha. Este texto de contenido fluye hacia el final de la linea, baja y repite hasta llenar la caja, como se explica en el apartado de Box Model y direcciones de texto.

Las propiedades de CSS para aplicar estilo al texto pueden clasificarse en dos categorias que veremos por separado:

  • Estilos del tipo de letra (font-styles): Afectan al texto como el tipo de letra, tamaño, negrita, cursiva, etc.
  • Estilos de disposición del texto (text layout styles): Afectan al espaciado y otras caracteristicas de disposición de texto.

Se aplica a todo el texto, no se aplica por partes de texto a menos que esten etiquetados con span y strong. Con pseudo elementos como ::firs-latter (primera letra del texto de un elemento), ::first-line (primera linea del texto de un elemento) o ::selection (selecciona el texto que esta resaltado por el cursor). Así se seleccionamos una parte del texto.

Tipos de letra (Fonts)

Veremos propiedades que afectan al tipo de letra (se aplican al texto).

Color

Establece el color del contenido de los elementos selecionados. Su valor puede ser cualquier unidad de color como palabras clave (ej, blue), codigos, etc. Tambien se puede aplicar a text-decoration (subrayados para el texto). El color negro es el estandar.

Ejemplo: si usamos el selector de elemento p con el color rojo, el contendio de este elemento se pondrá rojo.

Familia de tipos de letras

Usamos font-family para definir un tipo de letra diferente para el texto. Indica el tipo de letra (o una lista) para aplicar al texto de los elementos. El navegador le mostrará la letra seleccionada si los visitantes al sitio, la disponen en su computadora o dispositivo. De lo contrario, seleccionará la default font del navegador. Tambien podemos usar las fuentes de google, ningun dispositivo tendrá problemas.

Tipos de letras seguros para la web

Solo hay una cantidad limitada de tipos de letras disponibles en los dispositivos sin preocupaciones. Estos tipos de letras se llaman web safe fonts.

Los desarrolladores web desean tener un control sobre los tipos de letras para el contenido de texto. No hay manera de saber que fuentes disponen los dispositivos que van a acceder a la web. Pero todos los sistemas tienen tipos de letras seguras para la web. En Wikipedia hay un articulo que detalla una lista con fuentes seguras comunes en los sistemas, aunque pueden cambiar y evolucionar con los sistemas operativos.

  • Arial: Es un tipo de letra sans-serif. Tambien se podría añadir Helvetica que es más agradable.
  • Courier New: Es de tipo monospace, hay una más antigua que se llama Courier. Es buena practica añadir a las dos.
  • Georgia: Es de tipo serif, tiene una forma elegante y clasica de periodico, parecida a Times New Roman
  • Times New Roman: es de tipo serif, tiene una version más antigua llamada Times, es una buena practica agregar las dos.
  • Trebuchet MS: es sans-serif, este tipo de letra no suele estar en los dispositivos moviles, hay que tener cuidado al usarla.
  • Verdana: es sans-serif, con una estetica moderna muy agradable.

El sitio cssfontstack, tiene una lista para elegir tipos de letras seguros para distintos sistemas operativos.

Fuentes predeterminadas

CSS define cinco nombres genericos para los tipos de letra, serif, sans-serif, monospace, cursive, fantasy y fantasy. Son genericos y el tipo de letra exacto que se va a utilizar cuando se determinen, dependerá de cada navegador y varia en los sitemas operativos. El peor escenario es que el navegador tratará de proporcionar al menos una fuente que parezca apropiada. serif, sans-serif y monospace son predecibles, pero cursive y fantasy, no. El navegador debería proporcionar algo razonable, las dos ultimas son menos recomendables.

  • serif: Tipo de letra que tiene adorno en los extremos de los trazos, pueden ser redondeados o planos.
  • sans-serif: Letras que carecen de serifas (adornos)
  • monospace: Cada caracter tiene el mismo ancho, se usa en codigo de programación, en los editores, terminales de los OS.
  • cursive: intentan emular la letra manuscrita, con trazos fluidos y conectados
  • fantasy: Tipo de letra decoraciva.

Listas de tipos de letras

Ya que no podemos garantizar qué tipo de letra tienen en el sistema los que visitan la web, puedes proporcionar una lista de tipos de letras para que el navegador pueda elegir.

Con la propiedad font-family eleguimos como valores los tipo de letra y las separamos con una coma.

El navegador comienza al principio y busca el tipo de letra que esta disponible en el sistema o dispositivo.

Proporcionar un tipo de letra generico (serif, sans-serif y monospace) al final de la lista para que el navegador pueda proporcionar la go medianamente adecuado en el caso de que no haya ningun tipo de letra segura (Arial, Helvetica, Times New York, Verdana).

Tamaño de la letra

En el articulo unidades y valores revisamos las unidades de longitud y tamaño. El tamaño del tipo de letra (establecido con font-size) puede tomar estos valores en estas unidaes y porcentajes.

Pero las unidades más comunes para ajustar el tamaño del texto son:

  • px: (pixeles), numero de pixeles de altura para el texto. Es una unidad absoluta, su valor es intrínseco no depende de otro elemento.
  • em: 1em equivale al tamaño fuente que se haya establecido en el elemento que estamos diseñando, ej: si el elemento que diseñamos tiene font-size de 16px, será 1em (puede tener fuente heredada -cuando no la cambiamos-, se ve en el inspector); 2rem = 32px, haciendo la multiplicación, así sucesivamente. Puede ser complejo cuando hay muchos elementos anidados con diferentes tamaños de letra, pero podemos usar em para estatablecer el tamaño de todo, lo que facilita su mantenimiento.
  • rem: 1rem equivale al tamaño del elemento raiz de todo el documento es html que tiene 16px de tamaño de letra, entonces, 1rem es solo 16px; 2rem es 32px. Si redefinimos el tamaño de fuente del elemento raíz (HTML), en 10px; 1rem = 10px y 2rem = 20px, etc.

La font-size de un elemento se hereda del elemento raiz HTML que tiene 16px en todos los navegadores. Cualquier parrafo que no tenga definido un tamaño, va a heredar 16px del elemento HTML.

Las cosas se complican cuando se altera el tamaño de letra de los elementos anidados. ej: si en la etiqueta contenedor article con 1.5em que resulta 24px y luego quieres que los parrafos dentro tengan tamaño de 20px. Se necesitará establecer el tamaño en 20/24em que es 0.8,33em. Lo mejor es usar rem donde se pueda, simplificarán las cosas. Evitar establecer un tamaño de letra concreto para los elementos del contenedor. No es compatible con Windows 8 y anteriores.

Rem vs Em

Para tamaño de fuente (font-size) usamos rem para elementos que tienen que tener tamaños fijos y relativos al tamaño raíz. Cuando necesitamos que el tamaño de los elementos se adapte al cambio de tamaño del texto. usamos em. Por ejemplo, para valores de padding, margin, etc.

De otra forma practica, puede ir probando valores en em o rem y cambiando el tamaño de los elementos para ver cual unidad se adapta mejor a lo que desea.

Estilo y cuerpo del tipo de letra, efectos y decoraciones al texto

CSS proporciona cuatro propiedades para alterar el efecto visual y enfasis del texto:

  • font-style: Se usa para activar o desactivar la cursiva (se usa poco, a menos que necesites cursivas), tiene tres valores:
    • normal: Valor por defecto (desactiva la cursiva).
    • italic: Pone el texto en cursiva, si el sistema del visitante lo tiene, sino usará el estilo oblique
    • oblique: Inclina la version normal.
  • font-weight: Establece el gruso o peso del texto, tiene muchos valores y variantes (light, normal, extrabold, bold) pero los que se usan son normal y bold.
    • normal, bold: Grueso normal (defecto) y negrita
    • lighter, bolder: Gruego negrita, en un nivel por debajo (lighter) o por encima (bolder) del elemento padre.
    • 100-900: Valores numericos de negrita, permite un control más detallado que los valores en palabra.
  • text-transform: permite transformaciones sobre el tipo de letra, con los valores:
    • none: Impide la transformación.
    • uppercase: Pone todo el texto en mayus.
    • capitalize: Transforma las letras iniciales de cada palabra en el texto.
    • full-width: Transforma los caracteres para cada uno quede dentro de una caja de ancho fijo, similar a una fuente monoespaciada. Permite la alineacion de caracteres occidentales con los asiáticos.
  • text-decoration: decora el texto (principalmente se usa para desactivar el subrayado por defecto de los enlaces).
    • none: Desactiva la decoración presente del texto.
    • underline: Subraya el texto.
    • overline: Proporciona una linea arriba del texto.
    • line-through: Coloca un tachado sobre el texto.

Text-decoration es una propiedad abreviada de text-decoration-line por esto puede aceptar varios valores a la vez.

Textos sombreados

La propiedad text-shadow aplica sombras al texto, consta de 4 valores.

  • Desplazamiento horizontal: Admite la mayoría de las unidades y magnitudes pero se suele utilizar px Es obligatorio, va de der (pos) a izq (neg)
  • Desplazamiento vertical: Se comporta como el anterior y tambien es obligatorio, va de abajo (pos) hacia arriba (neg)
  • Radio de desenfoque: Cuanto más alto es este valor, mayor será la dispersión. Si no se incluye es 0. Las magnitudes soportadas son como las de los anteriores valores.
  • Color: Admite todos los colores soportados por CSS , su valor por defecto el negro sino se pone uno.

Nota: Los desplazamientos de las sombras aceptan valores negativos que cambian la dirección de la sombra.

Sombras multiples

Se pueden aplicar separando con una coma el grupo de valores que define cada sombra individual. Todo a un solo elemento.

Diseño del texto

Despues de las propiedades basicas para el tipo de letra, toca dar un vistazo a las propiedades para la disposición del texto.

Alineación del texto

La propiedad text-align se usa para mover el texto dentro de la caja, sus valores son los siguientes:

  • left: Alinea el texto por la izquierda.
  • right: Alinea el texto por la derecha.
  • center: Centra el texto.
  • justify: Varía los espacios entre las palabras para que todas las líneas de texto tengan el mismo ancho, cuando hay palabras largas en el texto hay que tener cuidado. Se puede combinar con la propiedad hyphens para dividir las palabras largas entre las líneas.

Interlineado

La propiedad line-height determina la altura entre cada línea de texto, admite los valores del apartado unidades y magnitudes, pero tambien puede tomar un valor sin unidades que actúa como multiplicador de font-size para determinar la altura entre líneas, es la mejor opción, como se ve en el ejemplo: 1.5 o 2 para doblar la altura de la fuente. Se usa para que el texto del body se vea mejor.

Espaciado entre letras y palabras

Las propiedades letter-spacing y word-spacing son utilies para lograr algun efecto o para aumentar la visibilidad de algun tipo de letra condensada.

Otras propiedades interesantes

Podrías explorar las siguientes para el tipo de letra:

  • font-variant: Cambia entre las opciones de versalita y normal.
  • font-kerning: Activa/Desactiva las opciones de interletraje
  • font-feature-settings: Activa/Desactiva diversas caracteristicas de los tipos de letras OpenType
  • font-variant-alternates: Controla el uso de los caracteres alternativos para un tipo de letra dado.
  • font-variant-caps: Controla el uso de los caracteres alternativos de las mayúsculas.
  • font-variant-ligatures: Armoniza la forma esteticas del texto.
  • font-variant-numeric: Modifica la estetica de la representación de los numeros.
  • font-variant-position: Controla como se ve el tamaño de los subíndices y superíndices.
  • font-size-adjust: Ajusta el tamaño visual del texto independientemente del tamaño de la fuente.
  • font-stretch: Cambia entre posibles versiones ampliadas de un tipo de letra determinado.
  • text-underline-position: especifica la posición de los subrayados.
  • text-rendering: Mejora la visualización del texto.

Para ver ejemplos en la documentación de MDN cambiar el idioma al ingles.

Otras propiedades para la disposión del texto:

  • text-indent: Afecta al espacio horizontal antes del comienzo de la primera línea del texto
  • text-overflow: Define como se va a mostrar el contenido que no entra en la caja.
  • white-space: Define como se usan los espacios en blanco y los saltos de linea asociados dentro del elemento.
  • word-break: Especifica si se deben dividir las palabras al final de la linea.
  • direction: Define la dirección del texto, como es un tema complicado, así que hay un apartado exclusivo. Es mejor dejarlo por defecto en un principio.
  • hyphens: Activa/Desactiva la separación de las palabras en idiomas permitidos.
  • text-align-last: Define la ultima alineacion de la ultima línea de un bloque o línea justo antes de un salto de línea forzado.
  • text-orientation: Define la orientación del texto en una línea.
  • word-wrap: especifica si el navegador puede dividir palabras al final de la línea para evitar desbordamientos del contenido.
  • writing-mode: Modifica las lineas de texto para que se puedan ver en horizontal (normal) o vertical.

Resumen de los fundamentos de texto y fuentes

La personalizacion del texto con CSS afectan a dos ambitos distintos: al tipo de letra que afecta a la letra misma y a la disposición del texto que afecta a como se ve el texto junto con los demás elementos de la web.

Para el tipo de letra tenemos: La propiedad color, family-font para definir nuestras fuentes favoritas en diferentes secciones de la web. Tambien las fuentes generica para que el navegador muestre una fuente apropiada cuando el cliente no dispone la fuente principal. Con family-font podemos crear una lista de fuentes así crear una experiencia adecuada. El tamaño de la fuente depende de diferentes unidades de valor, explicadas anteriormente con sus usos adecuados. Disponemos de un grupo de propiedades para estilizar el tipo de letra: como font-style (activa/desactiva cursiva, oblicua, etc.), font-weight (cambia el grueso del texto), text-transform (modifica las letras) y text-decoration (subrayados). De las sombras se encarga text-shadow hasta se pueden aplicar multiples sombras en diferentes direcciones.

Para la visualización del texto tenemos: La propiedad text-alin (diferentes alineaciones), line-height (interlineado/altura enter lineas de texto), letter-spacing (espacio entre letras) y word-spacing (espacio entre palabras).
