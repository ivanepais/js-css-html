Capitulo 2: Estilo a las listas

Las listas son como cualquier texto pero hay detalles y buenas practicas que hay que conocer.

Conociendo las listas de HTML

Hay tres tipos de listas: ordenadas (ol), desordenadas (ul) y las descriptivas (dl), esta tiene un encabezado/titulo (dt) y el parrafo (dd). Con las herramientas de desarrollador del navegador podemos ver como vienen predefinidas:

  • Las listas ul y ol tienen un margin superior e inferior de 16px (1em) y un padding-left de 40px (2.5em) que hace ver a la lista más alejada del borde que un parrafo u otros elementos, creando un espacio fuera (margin) y dentro (padding) de su caja.
  • los renglones (li) no tiene valores de espacio como margin, border y padding.
  • La lista dl tiene un margin superior e inferior pero no tiene padding.
  • El parrafo (dd) de la lista descriptiva tiene un margin-left de 40px (2.5em).
  • Un parrafo cualquiera tiene un margin superior e inferior de 16px (1em).

Las unidades em actuan con respecto al valor del tamaño de letra del mismo elemento.

Manejar el espaciado de la lista

Al diseñar listas es necesario que estas mantengan los mismos espacios verticales/top-bottom (tambien llamado ritmos verticales) que el resto de los elementos como parrafos e imagenes y el mismo espaciado horizontal/left-right entre uno y otro.

Ejemplo para el texto y su espaciado:

  • Como estilo general tomamos la etiqueta raíz HTML y se establece un tipo de letra y tamaño para todo el sitio que van a heredar los demas elementos.
  • Seleccionamos el titulo secundario, todas las listas y los parrafos para establecer su tamaño de fuente, así tendran el mismo espaciado superior e inferior que mantendrá el ritmo vertical constante.
  • Despues tomamos los renglones de las listas y los parrafos generales con un mismo interlineado (line-height) para mantener un mismo espaciado con un ritmo vertical consistente.
  • Ahora aplicamos el interlineado line-height para las listas descriptivas (dl) en su titulo (dt) y su parrafo (dd). Despues aplicamos negrita para el titulo y aumentamos el margen de abajo para su parrafo separandolo de los siguientes elementos, que es el mismo espacio entre los titulos y todos los parrafos, así mantenre la consistencia visual.

    • En resumen: para mantener el ritmo vertical consistente primero hacemos reglas para la fuente y los interlineados para todos los elementos en general y despues aplicamos estilo detallados para cada elemento.

Para esto necesitamos los mismos valores en font-size y line-height.

Estilo específicos de las listas.

Propiedades especificas para personalizar las listas ordenadas (ol) y no ordenadas (ul).

  • list-style-type: Las viñetas/iconos al comienzo del renglon pueden ser circulares, cuadradas para una lista no ordenada. Números, letras o números romanos para una lista ordenada.
  • list-style-position: Establece si las viñetas aparecen afuera o adentro al comienzo del renglón.
  • list-style-image: Imagen personalizada para las viñetas

Estilo de la viñeta

La propiedad que vimos antes, list-style-type permite establecer el tipo de viñeta, como por ejemplo: el valor upper-roman que muestra numeros romanos en mayús. El valor upper-alpha muestras las primeras letras del alfabeto en mayús. Hay muchos más ejemplos en la documentación.

Posición de la viñeta

La propiedad list-style-position puede mostrar las viñetas un poco más adentro o afuera del renglón. Por ejemplo, su valor inside muestra las viñetas dentro del renglon interrumpiendo el texto, es un poco desprolijo.

Imagen personalizada como viñeta

Podemos hacerlo con la propiedad list-style-image con el valor url() aunque es limitada en cuanto al control de la posición, tamaño, etc. Es conveniente usar la propiedad background que se detallo en la sección Diseño de cajas.

Siguiendo con el ejemplo del comienzo, para las listas no ordenadas ul reducimos el padding-left (desplazamiento del contenido dentro de la caja) de 40px a 20px. n esta misma regla sacamos las viñetas.

En otra regla tomamos la ul con li y le ponemos el mismo padding-left para que las líneas de texto se vean consistentes con las listas ordenadas y descriptivas para insertar imagenes de fondo.

Vamos a usar la propiedad background para controlar la las viñetas un una lista no ordenada. Tendremos en cuenta las siguientes propiedades:

  • background-image: Con su valor url() ponemos la dirección donde esta la imagen que usaremos como viñeta.
  • background-position: Podemos decidir en que lugar va a aparecer el elemento que seleccionamos. Usa las coordenadas x (horizontal), y (vertical) que empiezan en la parte superior izquierda de una caja. Por ejemplo, la posición 0 0 es justo el comienzo de un elemento.
  • background-size: Determina el tamaño de la imagen de fondo. Queremos que las viñetas sean de la misma altura que el renglón más o menos. El padding-left de 20px que hemos puesto sirve para que entre esta viñeta personalizada. Elegimos 1.6rem 1.6rem para el tamaño vertical y horizontal.
  • background-repeat: Tiene un valor por defecto que repite las imagenes de fondo, entonces le ponemos no-repeat para obtener una sola copia de la imagnen.

Propiedad abreviada para el estilo de listas

La propiedad list-style permite usar los valores de propiedad type, image y position. No hay orden y se puede usar uno, dos o los tres; estas tiene su valor por defecto que se pondrá cuando no determinemos una. Son los valores disc, nome y outside de las propiedades type, image y position.

Numeración de las listas

Para las listas ordenadas podemos tener numeros diferente numeración, que empieze inversamente o que sean letras del alfabeto; se puede lograrlo con una serie de atributos en la etiqueta de apertura del elementos ol.

Atributo Start

Después de ol podemos usar start="x" para empezar la lista con la numeración distinta de 1.

Atributo Reversed

Empieza la lista contando de mayor a menor, siguiendo con el ejemplo: start="x reversed.

Atributo Value

Se usa en los renglones (li) para poner los numeros que quisieramos, por ejemplo: li value="x".
